## 프로젝트를 통해 배운점
- React Framework를 활용하여 함수형 컴포넌트 방식으로 프론트 구성하는 방법
- Mobx를 사용해서 상태관리를 하는 방법
- Styeld components를 사용해서 컴포넌트를 CSS-in-JS 방식으로 스타일링 하는 방법
- 오픈API 를 활용하여 axios를 사용해 데이터를 가져오는 방법
- Material-UI 를 활용한 UI Framework 활용하는 방법
- intersection observer 를 활용하여 무한스크롤 적용하는 방법 
- YouTube Api 를 활용하여 예고편을 띄우는 방법
- 백엔드서버와 통신하는 방법
- 소셜로그인 토큰으로 받아 인증 / 인가 처리

## 실행방법
- Api.js 에 TMDB Apikey 등록

**1. 원격 저장소 복제**
<pre>
$ git clone https://github.com/jun111haha/movielog.git
</pre>

**2. 프로젝트 폴더로 이동**
<pre>
$ cd movielog
</pre>

**3. 필요한 node_modules 설치**
<pre>
$ npm install
</pre>

**4. 개발 서버 실행**
<pre>
$ npm run serve
</pre>
